# 🌐 Sito Web Aziendale

Benvenuto nel repository del **Sito Web Aziendale**.
Questo progetto rappresenta un sito moderno, responsive e performante progettato per presentare servizi, team e contatti di un’azienda.

---

## 🚀 Tecnologie utilizzate

* **Frontend:** React / JavaScript
* **Styling:** Tailwind CSS
* **Animazioni:** Framer Motion
* **Backend (opzionale):** FastAPI / Node.js
* **Deploy:** Vercel / Netlify

---

## 📂 Struttura del progetto

```
/public         # File statici (immagini, favicon)
/src
  /components   # Componenti riutilizzabili
  /pages        # Pagine del sito
  /assets       # Immagini e risorse
  /styles       # File CSS/Tailwind
```

---

## ✨ Funzionalità principali

* ✅ Design moderno e responsive
* ✅ Sezione Home con presentazione aziendale
* ✅ Sezione Servizi
* ✅ Sezione Progetti / Portfolio
* ✅ Sezione Contatti
* ✅ Animazioni fluide

## ⚙️ Installazione

Clona il repository:

```bash
git clone https://github.com/tuo-username/nome-repo.git
cd nome-repo
```

Installa le dipendenze:

```bash
npm install
```

Avvia il progetto:

```bash
npm run dev
```
