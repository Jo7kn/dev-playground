# NeuroX Chat Frontend

Il frontend dell'applicazione NeuroX Chat, una moderna interfaccia web per conversazioni con AI locali.

## 🚀 Avvio Rapido

```bash
npm install
npm start
```

L'app sarà disponibile su `http://localhost:3000`

## 🛠️ Tecnologie

- **React 18** con TypeScript
- **Tailwind CSS** per lo styling
- **Axios** per le API
- **Vite** come bundler

## 📦 Build

```bash
npm run build
```

## 🎨 Caratteristiche UI/UX

- Design glassmorphism con temi dinamici
- Gestione sessioni con sidebar
- Ricerca in tempo reale
- Auto-scroll intelligente
- Modali animate
- Responsive design

## 🔧 Configurazione

Il frontend si collega al backend su `http://127.0.0.1:7000`. Modifica `API_URL` in `App.tsx` per cambiare l'endpoint.

## 📱 Funzionalità

- Chat in tempo reale con AI locale
- Gestione multi-sessione
- Esportazione conversazioni
- Temi personalizzabili
- Ricerca messaggi
- Persistenza locale
